# CyberSim Facilitator Materials

Source files and working documents for CyberSim scenario and facilitator materials. This repo is where the files live during editing; finished PDFs are published as GitHub Releases for distribution.

## What's in here

Materials are organized by scenario and language. Each scenario contains some combination of:
- **InDesign source files** (`.indd`, `.idml`) — the editable originals
- **Linked assets** — images referenced by the InDesign files
- **PDF** — the current export, ready for distribution

## Editing and exporting

Edit the InDesign source. When ready to publish, export a PDF with the correct filename (see `RELEASES.md` for the naming convention) and follow the release process.

Do not rename PDF files between releases — the distribution links on the website use "latest release" URLs that depend on stable filenames.

## Publishing a new version

See **[RELEASES.md](RELEASES.md)** for the full workflow: how releases are versioned, how to publish to GitHub Releases, and how the website's download links work.

## Translations

New language versions can be added under a new language folder alongside the existing ones. Follow the same structure and use the ISO 639-1 two-letter language code in the PDF filename (e.g. `fr`, `es`, `ar`).
